# 🌍 Global Currency Converter (Java)

## 📌 Project Description
The Global Currency Converter is a simple Java-based console application that converts foreign currencies into Indian Rupees (INR). It currently supports conversion from:

- USD (US Dollar)
- EUR (Euro)
- GBP (British Pound)

The program takes user input for the amount and currency type, then calculates and displays the equivalent value in INR.

---

## 🎯 Why This Project Was Created
This project was developed to:

- Practice Java programming fundamentals  
- Understand conditional statements and methods  
- Learn user input handling using Scanner  
- Build a basic real-world utility application  

---

## ⚙️ Features / Utility
- Easy-to-use console interface  
- Supports multiple currencies  
- Fast conversion using predefined rates  
- Demonstrates modular coding (separate class for logic)

## 🚀 How to Run the Project

1. Save the file as:
   GlobleCurrencyConverter.java

2. Compile:
   javac GlobleCurrencyConverter.java

3. Run:
   java GlobleCurrencyConverter
